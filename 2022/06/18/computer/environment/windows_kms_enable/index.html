# windows 系统激活

[TOC]

# 须知

  1. 需要首先设置 kms 服务器，请确保 kms 服务器畅通。
  2. 安装对应版本秘钥激活。

# 卸载安装的密钥

slmgr /upk

# 安装密钥

slmgr /ipk M7XTQ-FN8P6-TTKYV-9D4CC-J462D

秘钥列表 https://docs.microsoft.com/zh-cn/windows-server/get-started/kms-client-activation-keys
2019 长期服务版秘钥 M7XTQ-FN8P6-TTKYV-9D4CC-J462D

# 测试 kms 服务商联通性

ping zh.us.to
kms.03k.org

# 设置 kms 服务商

slmgr /skms kms.03k.org

## 激活密钥
slmgr /ato

# 查看到期时间

slmgr –xpr

# 确认激活状态

slmgr -dlv
